# 学生网络结构图修改说明

## 输出文件

- 原始文件：`Manuscript/student_network_architecture_tensor.pptx`
- 修改后文件：`Manuscript/student_network_architecture_tensor_green_liquid.pptx`

## 修改目标

本次修改围绕四个目标展开：突出 BPCSCFC 中液态神经网络的时序建模能力，统一为绿色视觉体系，增强图形化表达，并减少说明性文字占比。

## 1. BPCSCFC 中的液态时序表达

原图中 BPCSCFC 更偏向“局部/全局切片 + Bi-CfC 分支”的模块框图，液态神经网络的连续时间特征不够直观。新版在 BPCSCFC 内部将四个分支表达为 `F_L CfC`、`F_G CfC`、`B_L CfC`、`B_G CfC` 四个液态 CfC 单元，并加入三类视觉符号：

- `h(t)` 节点链：表示隐状态随时间步推进。
- 循环箭头：表示液态神经网络的递归记忆能力。
- `Δt` 与点状轨迹：表示 CfC 对连续时间间隔和动态状态演化的建模。

这样 BPCSCFC 不再只是一个切片融合框，而是明确展示“正向/反向、局部/全局”四路液态时序建模后再融合为 `F_S^t`。

## 2. 绿色统一色调

新版整体使用绿色体系：

- 深绿色：主路径、关键模块和分类器。
- 中绿色：连接线、边框和张量图形。
- 浅绿色：模块背景和辅助区域。
- 黄绿色虚线：蒸馏相关 tap，例如 MemKD 和 TS-T。

这样主学生网络路径和辅助蒸馏路径仍可区分，但不会像原图一样出现蓝、紫、橙、绿多色并列导致视觉分散。

## 3. 图形化表达增强

新版减少了文字解释，将原来的文字标签替换为更直观的图形符号：

- 原始波形用柱状波形图表示。
- CNN 编码器用堆叠特征块表示下采样。
- `H_S` 与 `F_S^t` 用张量网格表示 `B×T_S×C`。
- BPCSCFC 中局部切片用竖向窗口条表示，全局切片用网格 stride 表示。
- DRASP 用 `G/L` 两个分支和统计柱状图表示 Global ASP 与 Local ASP。
- `Σ` 表示四路液态时序特征融合。

文字只保留模块名、关键变量维度和最小图例，便于论文图或汇报图中快速阅读。

## 4. 结构保留与信息压缩

新版仍保留原图的主要计算链：

`Raw waveform -> 1D CNN -> H_S -> BPCSCFC -> F_S^t -> DRASP -> Classifier -> Z_S`

同时保留蒸馏相关输出：

- `H_S -> MemKD`
- `F_S^t -> Z_S^t -> TS-T`

但将这些路径用虚线和小型模块表示，避免抢占学生网络主干的视觉权重。
